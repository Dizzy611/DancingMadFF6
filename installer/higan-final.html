Higan Installation Final Notes

The following is copied from the README's "Higan Post-Install" section. These steps must be followed in order to play the hack on Higan:


Due to the unique way in which Higan handles ROMs, a bit of messing around is required after you've finished installing the ROM hack.

First, you must load the ff3msu.sfc file in Higan (it will play the normal SPC music, this is expected at this point).

Then, you must close Higan, and open an Explorer window and go to %USERPROFILE%\Emulation\Super Famicom\ff3msu.sfc (paste it in the address bar at the top).

You must then move everything *but* the ff3msu.sfc file (that is, all the .pcm files, the manifest.bml, and ff3msu.msu) from the directory where you installed Dancing Mad to this directory. If you did this correctly, the ff3msu.sfc directory in %USERPROFILE%\Emulation\Super Famicom should have these files in it:

program.rom
manifest.bml
ff3msu.msu
and a number of .pcm files named ff3-##.pcm where ## is two numbers.


It might also have a save.ram file. It should *not* have an ff3msu.sfc file or any other .sfc file in it.

You can then load the game in Higan again as normal. You should now be hearing improved music through the MSU-1!
